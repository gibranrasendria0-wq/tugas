1. Mengapa closure sering digunakan untuk menyimpan data private?

Closure memungkinkan variabel disimpan di dalam function sehingga tidak bisa diakses atau diubah langsung dari luar. Data hanya dapat diakses melalui method yang telah disediakan. Hal ini membuat data lebih aman dan terkontrol.

2. Apa keuntungan menggunakan scope yang baik dalam program besar?

Scope yang baik membantu menghindari konflik nama variabel, membuat kode lebih rapi, mudah dibaca, dan mudah dipelihara. Selain itu, programmer dapat mengetahui dengan jelas di mana sebuah variabel dapat digunakan.

3. Apa risiko penggunaan hoisting yang tidak dipahami dengan baik?

Jika hoisting tidak dipahami, program bisa menghasilkan nilai undefined atau error seperti ReferenceError. Hal ini dapat menyebabkan bug yang sulit ditemukan karena urutan eksekusi program tidak sesuai dengan yang diperkirakan.

4. Mengapa Date Object penting dalam aplikasi nyata?

Date Object digunakan untuk mengelola tanggal dan waktu, seperti menghitung umur, mencatat waktu login, menentukan masa berlaku produk, membuat sistem absensi, jadwal acara, dan berbagai fitur lain yang berhubungan dengan waktu. Tanpa Date Object, pengolahan data waktu akan menjadi lebih sulit dan tidak efisien.
