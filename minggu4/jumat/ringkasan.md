API = Application Programming Interface
API adalah antarmuka yang memungkinkan aplikasi saling berkomunikasi. Bayangkan API seperti pelayan di restoran:
Kamu (client) memesan makanan ke pelayan
Pelayan membawa pesanan ke dapur (server)
Hasilnya diberikan kembali ke kamu
Kamu tidak perlu tahu cara memasak, cukup memesan dan menunggu hasil

Get   =  ngambil data
post  =  ngirim data baru
put   =  memperbarui semua data
patch =  memperbarui sebagian data
delete=  ngapus data
