1. events adalah aksi atau kejadian atau pristiwa yang terjadi di halaman web
2. Jenis-Jenis Event yang Sering Digunakan

Berikut adalah beberapa contoh **event** populer berdasarkan kategori pemicunya.

### Mouse Events

- **click**  
  Terjadi ketika elemen diklik.

- **dblclick**  
  Terjadi ketika elemen diklik dua kali secara cepat.

- **mouseover**  
  Terjadi ketika kursor mouse masuk ke area elemen.

---

### Keyboard Events

- **keydown**  
  Terjadi ketika tombol keyboard mulai ditekan.

- **keyup**  
  Terjadi ketika tombol keyboard dilepas.

---

### Form Events

- **submit**  
  Terjadi ketika formulir (*form*) dikirimkan.

- **change**  
  Terjadi ketika nilai dari input (seperti *dropdown* atau *checkbox*) berubah.

---

### Window / Document Events

- **load**  
  Terjadi ketika seluruh halaman selesai dimuat.

- **resize**  
  Terjadi ketika ukuran jendela browser diubah.
